Find DaMatch is a game developed by four students as a school group project. The game is a digital, single player, version of Spot It. The user must tap on the image that is found on both cards. Find DaMatch has several image sets, and gives the user the option of using custom image sets eithe from Flickr or directly from their phone. The user may change the difficulty which causes images to be rotated and/or resized. The project was given to us in iterations to simulate an agile development cycle, where each iteration was done in sprints. Each student took turns in various Scrum roles such as the team leader, product owner, and repo manager.
